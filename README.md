# eCanCrusher for Windows

## 使用方法

将待转换文件夹（其中包含 META-INF 文件夹、minetype 文件、OEBPS 文件夹）拖动至 eCancrusher 图标上，即可将文件夹转为对应的 epub 格式，生成的 epub 文件位于待转换文件夹相同路径下。
